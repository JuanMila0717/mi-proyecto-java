# Sistema de Ventas

## Autor
Juan Mila

## Descripción
Proyecto en Java conectado a MySQL usando DAO para manejar datos.

## Base de Datos
- Crear la BD: db_sistema  
- Importar: db_sistema.sql  

## Ejecución
- Abrir en NetBeans  
- Ejecutar pruebas en "presentacion"  

## Estructura
- database: conexión  
- datos: DAO  
- entidades: clases  
- presentacion: pruebas  

## Funcionalidades
- CRUD Cliente  
- CRUD Categoria
